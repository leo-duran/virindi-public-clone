This directory contains the VirindiViews port of SkunkVision. SkunkVision
was originally written by Gregory Kusnick. The C# frontend code in this
port was written by Virindi.

The original SkunkVision was released under the MIT license and is
available at:
http://sourceforge.net/projects/skunkworks

This plugin port uses the MetaViewWrapper classes for the Virindi View
Service. These wrappers are available at:
http://www.virindi.net/repos/virindi_public/trunk/ViewServiceConnector
